## Análisis realizado

El script *analisis.py* realiza:

- carga del dataset
- limpieza básica  
- cálculos para conocer:
  - temperatura máxima  
  - temperatura mínima  
  - temperatura promedio  
  - precipitación diaria  
- generación de otro CSV con esos valores  
- generación de un gráfico de temperaturas

Los archivos con los resultados estan en la carpeta /resultados

Se usaron las librerias Pandas y Matplotlib en Google Colab
